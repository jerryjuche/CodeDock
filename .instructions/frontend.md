# Frontend Engineering Standards

## Component Architecture
- Strict composition hierarchy
- Clear ownership boundaries
- Predictable state boundaries
- Co-locate related logic when maintainable

## Technical Requirements
- Accessibility-first
- Responsive by default
- Use design tokens exclusively
- Clean semantic layout
- Minimal nesting depth

## Forbidden
- Deep prop drilling
- Arbitrary spacing
- Inconsistent typography
- UI clutter
- Oversized containers
- Random shadows or gradients

## State Management
- Prefer local state first
- Clear data flow direction
- Predictable loading and error states

All frontend code must produce professional visual rhythm and maintainability.
