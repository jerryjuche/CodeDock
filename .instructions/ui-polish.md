# UI Polish Standards

## Reference Products
Linear, Vercel, Notion, Framer, Raycast.

## Required Qualities
- Clear visual hierarchy
- Restrained elegance
- Measured spacing
- Balanced negative space
- Consistent radii and elevation
- Precise alignment
- Intentional interactions

## Interaction States
- Consistent hover/focus/active/disabled/loading states
- Subtle micro-interactions

## Forbidden
- Flashy effects
- Over-animation
- Visual noise
- Competing focal points
- Cheap dashboard aesthetics

Every UI change must feel premium, intentional, and production-grade.
