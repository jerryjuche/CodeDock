# Code Review Standards

## Correctness
- Root cause addressed
- Edge cases covered
- No regressions introduced

## Architecture & Maintainability
- Patterns preserved
- Coupling minimized
- Contracts respected

## Quality
- Minimal diff
- Proper error handling
- Performance preserved
- Accessibility maintained

## Polish
- UI consistency
- Visual hierarchy
- Professional finish

Reject changes violating any instruction file.
